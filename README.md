A growing collection of hands-on benchmarks comparing Small Language Models (SLMs) running locally on consumer hardware.

Real-world tasks (story writing, summarization, extraction, planning)
Latency, token speed, and resource usage
Model stability across multiple runs
Practical constraints on laptops & edge devices
No cherry-picked results. No marketing. Just raw numbers.
